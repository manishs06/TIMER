# Fullscreen Timer & Task Manager

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)](https://timer-nu-livid.vercel.app/)

A beautiful, modern fullscreen timer application with integrated task manager for productivity and time tracking.

## What It Does

This application combines a powerful timer with an integrated task manager, allowing you to:

- **Time Tracking**: Set countdown timers or use a stopwatch to track your time
- **Task Management**: Add, complete, and manage tasks alongside your timer
- **Fullscreen Mode**: Display the timer in fullscreen for presentations, meetings, or workshops
- **Visual Feedback**: See progress bars and receive audio alerts when timers complete
- **Keyboard Controls**: Use keyboard shortcuts for quick and efficient control

## Features

- Countdown timer and stopwatch modes
- Integrated task manager with filters (All, Active, Completed)
- Fullscreen mode with hover controls
- Keyboard shortcuts for all functions
- Progress bar visualization
- Audio alerts when countdown completes
- Local storage for task persistence

## Usage

1. Set your timer using the hour, minute, and second inputs
2. Add tasks you want to complete during your timed session
3. Choose between Countdown or Count Up mode
4. Start the timer and work through your tasks
5. Use fullscreen mode (F key) for presentations or better focus

## Keyboard Shortcuts

- **Space** - Start/Pause timer
- **R** - Reset timer
- **F** or **F11** - Toggle Fullscreen
- **Esc** - Exit Fullscreen
- **C** - Switch between Countdown/Count Up mode
- **?** or **H** - Show/Hide keyboard shortcuts help
